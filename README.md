## API
Проект использует в своей работе собственное API для загрузки контента.

### Использование
```http request
GET /api/lab-content?lab=1 
```
где `lab=` - номер лабораторной работы.  

API возвращает объект TypeScript с HTML внутри следующего вида:
```typescript
{
  [questionNumber: string]: {
    question: string,      // Текст вопроса
    problem?: string,      // Дополнительное содержимое задачи (при наличии)
    answers: Array<{
      id: string,         // id ответа
      content: string     // Содержимое ответа
    }>
  }
}
```
Каждый пункт представляет собой HTML и может иметь как текст, так и изображения.

### Имена используемых файлов
API использует следующие имена файлов (представлены в репозитории):
- Вопросы: `Q{lab}{question}` (Q0101 для лабораторной 1, задания 1)
- Задачи: `P{lab}{question}` (P0102 для лабораторной 1, задания 2 с задачей)
- Ответы: `A{lab}{question}{answer}` (A010101 для лабораторной 1, задания 1, ответа 1 (неверные ответы заранее отсортированы))
